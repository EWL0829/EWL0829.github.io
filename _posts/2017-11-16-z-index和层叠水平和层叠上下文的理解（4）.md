z-index受制于层叠上下文，看代码看图
```
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Document</title>
    <style type="text/css">
        img{
            width: 300px;
        }
        .box{
            width: 450px;
            height: 450px;
            position: relative;
            background: lightpink;
            z-index: 0;
        }
        .box img{
            position: absolute;
            z-index: 10000;
        }
        .box1{
            width: 450px;
            height: 450px;
            position: relative;
            margin-top: -250px;
            background: lightblue;
            z-index: 1;
        }
        .box1 img{
            position: absolute;
            z-index: -10;
        }
    </style>
</head>
<body>
    <div class="box">
        <img src="../img/preview.jpg" class="img1">
    </div>
    <div class="box1">
        <img src="../img/preview.jpg" class="img2">
    </div>
    
    
</body>
</html>
```
